Formularios

Voy a crear un formulario con la etiqueta "form"

Voy a usar las etiquetas:
  1. "lengend" le da nombre al formulario
  2. "label" le da nombre al input siguiente
  3. "input" crea el input. Puede ser:
    1. number
    2. password
    3. text
    4. submit - este es para enviar
    5. email
    6. radio
    7. file
    8. etc...
  4. "select" es para crear la caja de seleccion
  5. "option" combinado con select, le da el nombre a las opciones
  6. "textarea" crea un area de texto que se puede agrandar y achicar
  7. "button" crea un boton. Es equivalente a "input - submit"
  8. "placeholder" pone texto en gris dentro de la caja
  9. "img" para poner una imagen. Se usa con "scr" donde se pone el link de la imagen
  10. "a href=" es para poner un link a una web
